Claude MCP

by Anthropic · cloud
reads & writes your datano key needed here

Claude connects to this planner through an MCP connector — so you can ask Claude about your day or tell it to capture something, from Cowork, the web, or your phone, and it works with the same data you see here.

youtalk to Claude→ ClaudeCowork · web · mobile→ planner-mcpSupabase edge function→ your datathis planner

What Claude can do

Readanytime

· today's agenda· goals· projects · habits· actions· notes · people· tracker / mood

Create & editon request

· goals· projects· habits · actions (tasks)· notes· people · log a metric· complete an action

It can never delete anything — deletions stay a manual action here in the app.

How it's wired

A Supabase edge function (planner-mcp) deployed next to the database. Claude authenticates with a secret token carried in the connector URL:

https://<project>.supabase.co/functions/v1/planner-mcp/<token>/mcp

The server-side key never leaves the server, and every query is locked to your account. Source and setup live in the repo under supabase/functions/planner-mcp.

Use it

In Claude → Settings → Connectors it's Personal Planner. Turn it on in a chat, then just talk: "what's on my plate today?" · "add a P1 task to call the dentist tomorrow" · "log my mood as 4".

Gemini

by Google · cloud · needs an API key
note sentimentvoice transcription

Gemini (gemini-3.5-flash) is a cloud model. Give it an API key with the button below and it powers two in-app features:

What it does

Note sentiment — reads each note and tags its emotional tone from a fixed vocabulary. Those tags color the note and drive the home notes widget's mood color.

Voice transcription (cloud option) — an alternative to on-device Whisper when you record a voice note; you choose the engine as you record.

Without a key these are simply off — sentiment is skipped and transcription falls back to Whisper. (Gemini also used to run the old in-app chat agent, now retired in favor of Claude MCP.)

Privacy

When used, the note text or the audio is sent to Google. If you'd rather nothing leave your device, skip the key and let Whisper handle transcription.

Whisper

OpenAI model · runs in your browser
on-deviceno key · no costprivate

Whisper (whisper-base) is a speech-to-text model that runs entirely in your browser via transformers.js — no API key, no per-use cost, and your audio never leaves your device.

What it does

It's the default engine for transcribing voice notes — record in the Notes composer and Whisper turns the audio into text, locally.

Good to know

The model (~75 MB, quantized) downloads once the first time you transcribe, then the browser caches it for every time after, including across reloads. The first transcription is slower while it downloads; after that it's quick and fully offline.
